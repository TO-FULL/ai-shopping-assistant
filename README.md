# AI 电商购物助手

智能理解您的购物需求，自动从淘宝、京东、拼多多筛选推荐最符合的商品。

## 功能特性

- **自然语言输入**：用日常语言描述购物需求，AI 自动解析
- **智能条件提取**：自动识别商品类型、价格、颜色、风格、材质、尺码等
- **多平台检索**：同时覆盖淘宝、京东、拼多多三大电商平台
- **AI 推荐理由**：每条商品都附有智能生成的推荐理由
- **灵活排序**：支持按相关度、价格、销量排序
- **快速标签**：提供常用购物场景一键搜索

## 快速开始

1. 克隆或下载本项目
2. 直接在浏览器中打开 `index.html`
3. 在输入框中描述您的购物需求，点击"智能搜索"

## 使用示例

```
我想买一条高腰微喇牛仔裤，黑色，春秋款，价格在100-200元之间，显瘦百搭
```

AI 将自动解析出：
- 商品类型：牛仔裤
- 颜色：黑色
- 风格：微喇、显瘦、百搭
- 季节：春秋
- 价格：100-200元

## 技术栈

- HTML5 / CSS3（原生，无框架依赖）
- Vanilla JavaScript（ES6+）
- 模拟电商 API + 智能匹配算法
